# LinkNest

LinkNest 是一个面向技术内容创作者与开发者的外链资源聚合与导航系统。该项目定位于为开源社区提供高质量、可分类、可检索的外部技术资源收录与展示方案，帮助用户快速定位所需的技术文档、视频平台、社区论坛及工具站点。LinkNest 适用于个人开发者构建私有书签导航、技术团队维护公共知识库、以及开源项目维护者整理外部参考链接等场景。

## 功能概览

- **多级分类体系**：支持自定义分类与子分类，可对收录的 URL 资源按技术领域、资源类型、适用人群等多维度进行组织，便于后续维护与扩展。

- **资源状态监控**：内置链接可用性检查机制，可定期检测已收录资源的访问状态，自动标记失效或响应超时的链接，辅助管理员及时清理或更新。

- **标签与全文检索**：为每个资源条目附加自定义标签，并支持基于标题、描述、分类路径的全文检索功能，提升海量链接的查找效率。

- **访问统计与热度排序**：记录每个外部链接的点击次数与最近访问时间，支持按热度、收录时间、字母顺序等多种方式排序展示。

- **导入与导出**：支持批量导入 URL 列表（支持纯文本、CSV 及 Markdown 列表格式），并可一键导出为 JSON、YAML 或标准 HTML 书签文件，便于迁移与备份。

- **嵌入与分享**：为每个分类或标签视图生成独立的嵌入代码段，允许将资源列表嵌入到其他文档页面或团队 Wiki 中，并支持生成短链接便于分享。

## 应用场景

- **技术团队内部知识导航**：开发团队可将常用的 API 文档、设计规范、CI/CD 工具地址、日志平台入口等统一收录至 LinkNest，新成员入职时即可通过该系统快速了解团队所使用的技术栈与工具链。

- **开源项目外部链接附录**：开源项目维护者可以使用 LinkNest 整理项目依赖的相关标准、参考实现、社区论坛及视频教程链接，作为项目文档的外部资源附录，方便贡献者与使用者查阅。

- **个人开发者的书签替代方案**：开发者可将日常浏览的技术博客、在线课程、沙盒环境、代码片段库等分散的浏览器书签迁移至 LinkNest，实现跨设备、跨浏览器的统一访问入口，并通过分类与检索功能提升使用效率。

- **技术社区资源聚合站**：技术社区运营方可基于 LinkNest 搭建垂直领域的资源导航站点，例如前端工具大全、机器学习数据集索引、开源监控方案汇总等，为社区成员提供结构化的外部参考信息。

## 快速开始

以下指令适用于 Linux / macOS / Windows WSL 环境，默认使用 Python 3.10 及以上版本。

```bash
# 克隆项目仓库
git clone https://github.com/your-org/linknest.git
cd linknest

# 安装依赖（推荐使用虚拟环境）
python -m venv venv
source venv/bin/activate  # Windows 下使用 venv\Scripts\activate
pip install -r requirements.txt

# 初始化数据库并启动开发服务
python manage.py migrate
python manage.py loaddata initial_categories.json  # 载入预置分类
python manage.py runserver --host 0.0.0.0 --port 8080
```

启动后，访问控制台地址 http://localhost:8080/admin 即可开始添加资源条目。默认管理员账户可通过 `python manage.py createsuperuser` 命令创建。

| Docker | 20.10+（可选） | 提供容器化部署方案，便于快速搭建生产级实例 |

## 常见问题

**问：LinkNest 是否可以完全离线运行，不依赖外部网络？**

答：LinkNest 的核心功能（分类管理、检索、导入导出、访问统计）完全可在内网环境运行，无需访问外部网络。唯一的网络依赖是链接状态检测功能，该功能会向外发送 HTTP 请求以验证资源的可达性。若在内网使用，可通过配置将检测功能关闭或设置为仅检测内网地址。

**问：如何迁移已有的大量浏览器书签到 LinkNest？**

答：LinkNest 支持从浏览器导出的 HTML 书签文件（所有主流浏览器均支持此格式）以及通用 CSV 格式进行批量导入。您可在管理后台的“导入”界面选择对应文件，系统会自动解析并映射分类。若书签数量超过 500 条，建议使用命令行脚本 `scripts/import_batch.py` 进行异步导入，以避免请求超时。

**问：是否支持多用户协作编辑资源列表？**

答：LinkNest 内置基于角色的访问控制（RBAC），管理员可创建普通编辑者与只读访客账户。每个资源条目可单独设置编辑权限，分类层面可配置可见范围。您可以通过管理后台的“用户与权限”面板进行精细控制，满足团队协作场景需求。
